Para este ejercicio tenemos que hacer un menu que controlemos incidencias que tienen los ciclistas durante su carrera. El primer metodo que hacemos es el de insertar pedimos los valores de id, numero de etapa y el tipo de incidencia, posteriormente comprobamos si el id del ciclista introducido existe, si existe verificamos la etapa y si tambien existe introducimos los valores a la tabla. Para el segundo lo unico que hay que hacer es scar todos los valores que hay en la tabla y mostrarlos por pantalla y por ultimo hay que hacer exactamente lo mismo pero pidiendo el id de un ciclista y mostrando sus incidencias.
